
# Electronic Dictionary

## Описание проекта

Electronic Dictionary — это настольное приложение на Java с графическим интерфейсом (Swing), которое позволяет:

- переводить английские слова на русский язык;
- добавлять новые слова в словарь;
- сохранять слова в JSON файл;
- просматривать историю поиска;
- работать с пользовательскими исключениями;
- использовать основные принципы ООП.

Проект разработан как курсовая работа по теме объектно-ориентированного программирования (OOP) на языке Java.

---

# Используемые технологии

- Java
- Java Swing
- OOP
- Collections Framework
- Exception Handling
- File I/O
- JSON (ручная реализация)
